## Installation

1. Clone the repository:
```
git clone https://github.com/HansLeon-stack/Apple_website.git
```

2. Navigate into the project directory:
```
cd Apple_website
```

3. Install dependencies:
```
npm install
```

4. Run the development server:
```
npm run dev
```
